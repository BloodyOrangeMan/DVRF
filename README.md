# 介绍

DVRF 的全称是 Danm Vulnerable Router Firmware，该项目是一个基于 [OpenWrt](https://openwrt.org/) 改造的漏洞固件。用 CTF 模式来帮助安全专业人员测试物联网设备中常见的漏洞，其中部分漏洞题基于公开的 CVE 漏洞。
